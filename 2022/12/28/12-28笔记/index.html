JavaSE基础语法

2022/12/28 学习 Java

6.Java语言 语法
　　运算符
　　　1.算术运算符 + - * / %（注：除法中两个操作数都是整数，结果也是正数，做整除运算，如果其中一个不是整数则不是整除）
　　　2.比较运算符
　　　3.逻辑运算符，参与运算的操作数都是Boolean型的，其结果也是Boolean型
　　　4.赋值运算符 注：a++（先取值后自增）++a（先自增后取值）
　　　5.位运算符 不讲，按bit运算
　　　6.条件运算符 max = a > b ? a : b;
　　　7.其他

　　优先级：小括号>单目运算符>算数>比较>逻辑(与或)>赋值
　　　单目：只需一个操作数！，+，-，++，–等

　　表达式：操作数和运算符组合到一起的式子。

　　语句：由分号结束的完整的指令
　　　在Java中必须以英文状态下的分号结束;;;;;;;;

　　流程控制
　　　一、顺序结构：按照顺序依次执行
　　　二、选择结构，分支结构：
　　　　1.单分支 if（）{}
　　　　2.双分支 if（）{} else {}
　　　　3.嵌套多分支 if（）{ if（）{ } }
　　　　4.switch结构
　　　三、循环结构
　　　　while(表达式){循环体}：先判断后循环，比较适合没有循环变量的场景
　　　　do{循环体}while(表达式)：先循环后判断，至少执行一次循环体
　　　　for循环：for(表达式1;表达式2;表达式3){循环体},表达式1和表达式3可以没有，但是分号不能没有。特别适合有循环变量的场景

package com.yuan;  
import java.io.*;  
import java.util.*;  
public class Test2 {  
    public static void main(String[] args) {

        int x = 10;// 顺序结构
        int y = 20;
        System.out.println(x + y);

        if (x > y) { // 括号里的结果是Boolean值(true或false)
            System.out.println("a大于b");
        } else if (y > x) {
            System.out.println("b大于a");
        } else {
            System.out.println("a等于b");
        }

        int year = 2000; // 判断闰年
        if (year % 400 == 0) { // 1、能被4整除,但不能被100整除 2、能被400整除
            System.out.println(year + "是闰年");
        } else if (year % 4 == 0) {
            if (year % 100 == 0) {
                System.out.println(year + "不是闰年");
            } else {
                System.out.println(year + "是闰年");
            }
        } else {
            System.out.println(year + "不是闰年");
        }

        int sum1 = 0;
        for (int counter = 1; counter <= 100; counter++) {
            sum1 = counter + sum1;
        }
        System.out.println(sum1);

        int i = 100;
        while (i < 1000) {
            int a = i / 100;
            int b = i % 100 / 10;
            int c = i % 10;
            int temp = a * a * a + b * b * b + c * c * c;
            if (temp == i) {
                System.out.println(i);
            }
            i++;
        }

        int sum = 0;
        for (int q = 1; q <= 100; q++) {
            sum += q;
        }
        System.out.println(sum);
    }
}
