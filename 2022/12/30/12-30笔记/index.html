JavaSE基础语法

2022/12/30 学习 Java

函数递归调用
　　　如果一个函数，直接或间接的不断调用自身，就是递归调用
　　　所有的递归，本质上都可以用循环来解决
　　　数据结构，函数调用后会向栈里添加这个函数，不断添加会导致栈溢出，StackOverflowError

　　　　　public static void sayHello() {
　　　　　　　System.out.println(“hello”);
　　　　　　　sayHello();// 一直递归，栈溢出
　　　　　}

　　　递归，如果递归没有退出条件就会成为死循环
　　　　　//计算n的阶乘
　　　　　public static long jie(int n) {
　　　　　　　if (n == 1) {
　　　　　　　　　return 1;
　　　　　　　}
　　　　　return jie(n - 1) * n;
　　　　　}
　　　
　　　递归案例：汉诺塔游戏

　　　　　public static void main(String[] args) {
　　　　　// 递归案例
　　　　　// 汉诺塔案例
　　　　　　　move(3, ‘A’, ‘B’, ‘C’);
　　　　　}

　　　　　// 将n个盘子，从x移动到z上，借助于y
　　　　　public static void move(int n, char x, char y, char z) {
　　　　　　　// 判断是不是最后一步，最后一步直接移动一个就完成
　　　　　　　if (n == 1) {
　　　　　　　　　System.out.println(x + “->” + z);
　　　　　　　　　return;
　　　　　　　}
　　　　　　　// 第一步
　　　　　　　move(n - 1, x, z, y);
　　　　　　　// 第二步
　　　　　　　System.out.println(x + “->” + z);
　　　　　　　// 第三步
　　　　　　　move(n - 1, y, x, z);
　　　　　}

String类的使用
　　字符串的属性和行为
　　　1.length()获取字符串的长度
　　　　　System.out.println(a.length());

　　　2.charAt()函数，用于获取指定位置的字符
　　　　　System.out.println(a.charAt(2));

　　　3.compareTo()函数，用于与另一个字符串比较大小,注意是大小，0表示相等，负数表示小于
　　　　　String c=”ab”;
　　　　　System.out.println(a.compareTo(c));

　　　4.compareToIgnoreCase()比较大小时，忽略大小写

　　　5.concat()函数用于将两个字符串拼接到一起
　　　　　System.out.println(a.concat(c));

　　　6.contain(),判断是否包含另外一个字符串，返回boolean值

　　　7.contentEquals(),用于判断，两个字符串内容是否相等

　　　　equals(),可以判断两个项目是否相等
　　　　equalsIgnoreCase(),比较时忽略大小写

　　　　==仅用于判断两个基本数据类型的值是否相等
　　　　如果使用==判断两个引用类型的话，则表示判断两个引用是否指向同一个对象

　　　8.startsWith()或endWith(),判断字符串是否以某个字符串开始或结束，返回boolean值

　　　9.indexOf(),返回某个字符串在字符串中首次出现的位置，从0位置开始，找不到返回-1
　　　　lastIndexOf()，从后向前查找，但返回的值仍是从前向后的排序
　　　　　System.out.println(a.indexOf(“abb”));

　　　10.isEmpty(),用于判断是否空字符串
　　　　　System.out.println(a.isEmpty());
　　　 isBlank(),判断是否是空白字符串，包括空格换行等都是空白

　　　11.repeat(),将一个字符串重复指定的次数
　　　　　System.out.println(c.repeat(3));

　　　12.replace(,)替换字符串，把前者替换成后者

面向对象
　　　是一种编程思想，而不是一种语言，或者技术。
　　　分块解决，最后组合起来解决大问题。
　　　适合解决大的场景，宏观问题。

　　　面向过程：将待解决的问题分解成一系列步骤（算法），分别执行这些步骤，执行结束，问题则解决。
　　　比较适合小规模或者局部的问题。

　　　什么是对象？
　　　　　万物皆为对象。
　　　　　对象，即具有静态属性和动态行为的个体。
　　　　　没有两个对象是完全相同的。
　　　什么是类？
　　　　　分类、类别、类型。
　　　　　白马非马（白马：某一匹白色的马，对象，马：类 ）
