Java基础

2022/12/27 学习 Java

1.JDK和JRE
　　JDK：java development kit，Java开发工具包
　　JRE：Java runtime environment，Java运行环境

2.Java/jdk的版本
　　Java是门语言，jdk是开发工具包，虽然不一样但是对于版本经常一起说，理解成一个也可以
　　　1.0，1.1，1.2（里程碑），1.3，1.4，1.5（重要的里程碑），1.6，1.7，1.8（里程碑）
　　　James的Sun卖出，被Oracle收购，
　　　9，10，11，12，13，14，15，16，17（LTS：long term support，长持支持版本），19

3.Java历史
　　James基于c++改造而来，原本叫做Oak（橡树）
　　Sun公司卖出，Oracle收购

4.jdk的安装
　　1.安装jdk。执行安装程序jdk-17_windows-x64_bin.exe
　　2.安装路径：尽量不要用到中文、空格、特殊字符等
　　3.设置环境变量：
　　　　　右键我的电脑-属性-高级系统设置-环境变量
　　　　　在系统变量中添加 变量名：JAVA_HOME 变量值：D:\Environment\jdk\jdk-17.0.5
　　　　　在Path中编辑，新建一个 %JAVA_HOME%\bin
　　　　注：bin：binary二进制，conf：configuration配置文件

5.IDE:集成开发环境
　　eclipse：免费，IBM
　　idea：社区版和旗舰版（付费）
　　net beans：

　　配置eclipse环境
　　　1.配置工作环境utf—8编码：window->preferences->workspace->Text file encoding->UTF-8
　　　2.配置已安装的jdk：window->preferwnces->java->Installed JREs->把其他没用的都删除，然后添加新的

　　package：包，来管理类（命名 一般域名的倒写）
　　　　Action类所在包命名规范：公司名称.开发组名称.项目名称.web.struts.action
　　　　　例如：net.linkcn.web.struts.action
　　class、interface：类、接口　
　　　　类　命名规范：以大写字母开头，如果有多个单词，每个单词头字母大写
　　　　　例如：StudentInfo
　　　　接口　命名规范：以大写字母“I”开头，如果有多个单词，每个单词头字母大写
　　　　　例如：IStudentInfo

　　快捷键：
　　　1.ctrl+D 删除光标所在行
　　　2.ctrl+S 保存
　　　3.ctrl+F 英文状态下可以格式化代码，但是中文状态下会切换简繁体

　　eclipse项目导入导出
　　　导入：Import->General->Existing projects into workspace
　　　导出：项目右键->show in->System Explorer，在文件夹内显示然后将文件夹复制即可

6.Java语言 语法
　　基础语法：
　　　1.变量、常量
　　　2.数据类型:分为两大类，基本数据类型和引用类型

　　8大基本数据类型 　　　
　　　byte：整型，占1个字节。-128——127，可能用于文件读写
　　　short：整型，占2个字节，-32768——32767，几乎不用
　　　int：整型，占4个字节。-2147483648——2147483647
　　　long：整型，占8个字节，范围很大
　　　注：long f = 22222222222222L;
　　　f 会被默认为是int，需要在最后加 L 强制指定long类型
　　　运算中，在任一个数后加L，即可结果为long类型

　　　float：单精度小数，４个字节，可以精确到小数点6-7位
　　　double：双精度小数，８个字节，可以精确到小数点16-17位
　　　注：小数自动默认为是都是double类型的值，float d1 = 3.14 报错，可以改成 flout d1 = 3.14F,在最后加 F 将3.14强制改为float类型，（常值的赋值需要加 F）
　　　但要区别于变量赋值 double d2 = d1（此时非常量参数赋值：自动类型转换），小可以改大
　　　但float d3 = d2 错误，应改成 float d3 = （float）d2，大改小不合法，进行强制类型转换

　　　char：字符型 两个字节，本质是保存的整数
　　　boolean：布尔型 true false
