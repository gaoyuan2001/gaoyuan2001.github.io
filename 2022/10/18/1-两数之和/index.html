1.两数之和

2022/10/18 学习 算法 数组 LeetCode

题目：
给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target 的那 两个 整数，并返回它们的数组下标。

你可以假设每种输入只会对应一个答案。但是，数组中同一个元素在答案里不能重复出现。

你可以按任意顺序返回答案。

示例 1：
输入：nums = [2,7,11,15], target = 9
输出：[0,1]
解释：因为 nums[0] + nums[1] == 9 ，返回 [0, 1] 。

示例 2：
输入：nums = [3,2,4], target = 6
输出：[1,2]

示例 3：
输入：nums = [3,3], target = 6
输出：[0,1]

来源：力扣（LeetCode）
链接：https://leetcode.cn/problems/two-sum

自己的思路和解法：
思路：1、嵌套for循环进行查找；2、不能重复出现相同的答案，要注意(1,2)和(2,1)不一样；3、最后返回值是直接返回空数组还是返回null。

class Solution {
    public int[] twoSum(int[] nums, int target) {        
        int len = nums.length;                       //形参len：数组长度
        for(int i = 0; i < len; i++) {               //嵌套for循环
            for(int j = 0; j < len; j++) {
                if(i == j)                           //因为每次 j的循环 都要全部循环，当i = j时需要跳过
                    continue;                        //这里区别break，（1，2）和（2，1）一样吗                
                if (nums[i] + nums[j] == target) {   //查找两数之和=target
                    int[] result = {i,j};            //给result初始化及赋值，并用作返回
                    return result;                   //返回result
                }
            }            
        }
        return null;                                 //如果没有查找到则会返回null，
                                                     //我在这不返回空result数组的想法是：感觉初始化一个数组占空间，但是在实际应用中还是返回空数组，否则在客户段调用时需要进行数组的非空判断，并且有时调用内置api还容易报错
}
