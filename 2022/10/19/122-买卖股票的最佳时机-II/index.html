122.买卖股票的最佳时机 II

2022/10/19 学习 算法 数组 LeetCode

题目：
给你一个整数数组 prices ，其中 prices[i] 表示某支股票第 i 天的价格。

在每一天，你可以决定是否购买和/或出售股票。你在任何时候 最多 只能持有 一股 股票。你也可以先购买，然后在 同一天 出售。

返回 你能获得的 最大 利润 。

示例 1：
输入：prices = [7,1,5,3,6,4]
输出：7
解释：在第 2 天（股票价格 = 1）的时候买入，在第 3 天（股票价格 = 5）的时候卖出, 这笔交易所能获得利润 = 5 - 1 = 4 。
随后，在第 4 天（股票价格 = 3）的时候买入，在第 5 天（股票价格 = 6）的时候卖出, 这笔交易所能获得利润 = 6 - 3 = 3 。
总利润为 4 + 3 = 7 。

示例 2：
输入：prices = [1,2,3,4,5]
输出：4
解释：在第 1 天（股票价格 = 1）的时候买入，在第 5 天 （股票价格 = 5）的时候卖出, 这笔交易所能获得利润 = 5 - 1 = 4 。
总利润为 4 。

示例 3：
输入：prices = [7,6,4,3,1]
输出：0
解释：在这种情况下, 交易无法获得正利润，所以不参与交易可以获得最大利润，最大利润为 0 。

来源：力扣（LeetCode）
链接：https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-ii

自己的思路和解法：
思路：1、因为是股票可以一直持有，所以可以按阶段来算，算出每个阶段的利益，加和；2、也可以每天进行买入卖出，但需要先判断利润的正负；3、可以画折线图来理解。

class Solution {
    public int maxProfit(int[] prices) {
        /*第一种方法：这个算法主要是以阶段来划分，从一个最低值买入，一直持有，直到出现最大值卖出，此阶段完成
        在寻找下一个阶段，直到日期结束*/

                i++;
            }
            min = prices[i];
            while(i<length-1 && prices[i]<prices[i+1]){
                i++;
            }
            max = prices[i++];      //如果这里变成i，会报响应时间超时，i++可以理解为此时已知prices[i]大于prices[i+1]，直接进入下一阶段就可以，先赋值在自加
            incom = incom+max-min;
        }
        return incom;
        */
        
        /*第二种方法：每天都会模拟买入卖出，如果今天卖出价高于昨天买入价，即盈利，实际也卖出（反之亏损，则不卖出），卖出后会接着买入今天的，把盈利之和算出，即为总盈利*/
        if(prices==null || prices.length==0)
            return 0;
        int length =prices.length,incom =0;
        for(int i=0;i<length-1;i++){      //这里的i的初始值要和最大值协调好，因为下面的prices[i+1],这个i+1，不能超出length，如果改为prices[i]-prices[i-1],这里的i=1,i<length
            int temp =prices[i+1]-prices[i];
            if(temp>0)
                incom+=temp;
        }
        return incom;
    }
}
