定位：可以破坏默认的文档流规则

可以将一个元素定位到网页中的任意位置

position：位置，static|absolute|relative

top，left，right，bottom

  • static：无定位

  • absolute：绝对定位，

    参照物的规则：以当前元素，最接近的非static定位的祖先元素作为参照，如果没有，则以body为参照

  • relative：相对定位

    参照物规则：以当前元素，未偏移前的自身的位置为参照

    注意事项：相对 定位，如果不指定偏移，则位置不变。往往用于作为其他绝对定位为参照物

  • fixed：固定定位

    参照物规则，以可视区域为参照，页面滑动，可以固定不动

z-index：层序，谁在谁的上面，垂直于显示器为z轴，设置数值，一般取-10万—10万

<style type="text/css">
            .my {
                width: 200px;
                height: 100px;
                background-color: pink;
                /* 绝对定位 */
                position: fixed;
                
                top: 200px;
                left: 200px;
                /* right: 200px;
                bottom: 200px; */
            }
            .parent {
                width: 300px;
                height: 200px;
                background-color: gray;
                /* position: absolute;
                top: 100px;
                left: 100px; */
                margin-top: 100px;
                margin-left: 100px;
            }
        </style>
    </head>
<body>
    <div class="parent">	
        <div class="my"></div>
    </div>
</body>

样式表的层叠，继承和优先级

层叠：一个元素的样式，可能是多种样式修饰叠加的效果。

继承：某些样式具备继承特性。一般是文字格式化样式（字号，字体……）

优先级：参考规则

  • 1.样式修饰离修饰元素越近，优先级越高

  • 2.选择器选择的越精准（选择器的名称越是精准定位到哪一个元素），优先级越高

  • 3.!important优先级非常高。比行内优先级高

  • 4.max-width/min-width/max-height/min-height：优先级高

<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title></title>
        <style type="text/css">
            #my {
                background-color: black;
            }
            .my {
                width: 300px;
                height: 200px;
                background-color: gold;
            }
            
            div {
                background-color: pink !important;
                max-width: 300px;
            }
            
            body>div.my {
                width: 1000px !important;
            }
        </style>
    </head>
    <body>
        <div id="my" class="my" style="background-color: green;">你邓</div>
    </body>
</html>

额外的布局：flex（弹性布局），css3的万能布局

弹性布局：display：flex；

主轴，交叉轴（默认横向为主轴）

flex-direction：row横向为主轴，column纵向为主轴

justify-content：flex-start/center/end主轴的对齐方式，space-between，space-around环绕

align-items：交叉轴对齐方式

flex-grow：1自动增长，增长速度为1

align-self：设置自己的单独的对齐方式

<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title></title>
        <style>
            .parent {
                width: 800px;
                height: 400px;
                border: 2px solid #ccc;
                margin: 0 auto;
                /* 弹性布局 */
                display: flex;
                /* 如果值为row，则横向为主轴，如果值为column，则纵向为主轴 */
                flex-direction: row;
                
                /* 主轴对齐方式 */
                justify-content: center;
                
                /* 交叉轴对齐方式 */
                align-items: center;
            }
            .parent>div {
                height: 100px;
                width: 100px;
                background-color: pink;
                margin-right: 10px;
                flex-grow: 1;
            }
            .parent>div:last-child {
                margin-right: 0;
            }
            .parent>div:nth-child(2) {
                /* 对单个元素设置对齐方式 */
                align-self: flex-start;
                flex-grow: 4;
            }
        </style>
    </head>
    <body>
        <div class="parent">
            <div></div>
            <div></div>
            <div></div>
            <div></div>
        </div>
    </body>
</html>
