固定套路：

  • 1.引入相关库。注册驱动

    注册驱动，看看有没有这个类，如果没有报错

    try {
    Class.forName("com.mysql.cj.jdbc.Driver");
} catch (ClassNotFoundException e) {
    throw new RuntimeException(e);
}

  • 2.获取数据库连接

url、user、password

然后用Connection类创建对象，以创建与数据库之间的连接

url：mysql：是数据库类型，也可以是Oracle等

　　localhost：3306是地址，这里用的是本地地址，3306是接口名，与数据库一直，也可以写其他电脑的IP

　　student2022：是数据库的名字

user：mysql的用户名，这里要注意此用户有没有读写的权限

passwor：上面用户名的密码

String url ="jdbc:mysql://localhost:3306/student2022?" +

  • 3.创建statement

    分成两种操作。一、读操作 二、写操作
    创建一个sql载体，用于装载sql语句

    stmt：相当于一个卡车，用于把sql语句运送到数据库并返回一定的数据，运输的路就是conn创建的连接

    Statement stmt = conn.createStatement();
String sql = "select id, stu_id, name as 姓名, sex, birthday, phone from t_student where id < 10";

  • 4.发送sql进行查询操作，返回结果集

    发送sql到数据库服务器，返回一个结果集

    executeQuery(sql)：mysql中的select，返回一个结果集

    executeUpdate(sql)：mysql中的update、insert、delete，返回一个int整数值，表示改动了几行数据

    ResultSet rs = stmt.executeQuery(sql);

完整代码：

import java.sql.*;

/**
 * @Title: Select
 * @Project: lesson0208
 * @Description:
 * @Author: 14508
 * @Version:
 * @Creat: 2023/2/8 20:48
 */
public class Select {
    public static void main(String[] args) {

        //1.注册驱动，看看有没有这个类，如果没有报错
        try {
            Class.forName("com.mysql.cj.jdbc.Driver");
        } catch (ClassNotFoundException e) {
            throw new RuntimeException(e);
        }

        //2.获取数据库连接
        String url ="jdbc:mysql://localhost:3306/student2022?" +
                "characterEncoding=UTF-8&useSSL=false&autoReconnect=true&" +
                "serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true";
        String user ="root";
        String password = "root";
        Connection conn = null;
        try {
            conn = DriverManager.getConnection(url, user, password);

            //3.分成两种操作。一、读操作 二、写操作
            //读操作
            //创建一个sql载体，用于装载sql语句
            Statement stmt = conn.createStatement();
            String sql = "select id, stu_id, name as 姓名, sex, birthday, phone from t_student where id < 10";

            //4.发送sql到数据库服务器，返回一个结果集
            ResultSet rs = stmt.executeQuery(sql);

            //5.从结果集中取数据
            while(rs.next()) {
                int id = rs.getInt("id");
                String stuId = rs.getString("stu_id");
                String name = rs.getString("姓名");
                String sex = rs.getString("sex");
                Date birthday = rs.getDate("birthday");
                String phone = rs.getString("phone");

                System.out.println(id +" "+ stuId + " " + name + " " + sex + " "+birthday + " " + phone);
            }
        } catch (SQLException e) {
            throw new RuntimeException(e);
        } finally {
            //6.关闭数据库
            if(conn != null){
                try {
                    conn.close();
                } catch (SQLException e) {
                    throw new RuntimeException(e);
                }
            }
        }

    }
}
