续-Js语法

2023/2/20 学习 前端

DOM：Document Object Model 文档对象模型

在js中，将每一个html元素都视作一个对象。通过js就可以操作这些对象

有键值对，即对象

my.style.backgroundColor = “pink”//键值对， backgroundColor就是一个key，把value“pink”赋给key
my.style[“backgronud - color”] = “pink”;

在js钟，有一些类型，拥有和数组类似的行为（可以用下标索引，有length属性），称之为类数组

类数组，HTMLCollection类型的对象

获取dom元素的方法

1.document.getElementById();//不用

2.document.getElementByClassName();//不用

3.let my = document.querySelectAll()//在dom上下文，去获取所有匹配的元素，返回NodeList对象//(#idname)(.classname)(“div”)

4.let my = document.querySelect()//在dom上下文，只会获取第一个匹配的元素

dom对象常用属性和行为

1.innerText：指代元素的内容，可读可写

2.innerHtml：指代元素的Html内容，可读可写

b.name 和 b[“name”]一样，都是b对象通过key—name，取得value

3.style：指代元素行内样式

4.className：读写dom元素的class值

5.tagName：读出dom元素的标签值，返回一个标签名，只可读

6.classList.：用于指代dom元素的class值的一个对象，进行类名增加/删除等，add();romove();

7.removeAttribute(“属性名”)：用对象调用，参数为属性名称，删除对象的一个属性

8.hasAttribute(“属性名”)：用于判断是否拥有某个指定名称的属性

9.getAttribute(“属性名”)：获取某个属性的值

10.setAttribute(“属性名”,”要给属性名的值”)：用于设置（增加）属性

11.增删改查：

查：query：

  • document.querySelectAll()
  • document.querySelect()

增：

  • createElement(“li”)：凭空创造一个新的dom元素
  • parent.appendChild(child)：父dom追加一个字元素（child）
  • parent.insertBefore(newLi,lastLi)：可以指定添加在某一个元素之前
let u = document.querySelector("ul");

let newLi = document.createElement("li");

newLi.innerText = "5";

u.appendChild(li)；//追加已有的元素，相当于转移，追加没有的元素，相当于增加

删：domElement.romove();//dom元素移除自身

parent.removeChild(child);用父dom移除一个指定的子元素

12.四向遍历：

  • 向上查找

    parentElement：获取父元素，唯一

  • 向下查找

    childElementCount：获取子元素的个数，只读

    children：获取所有子元素，返回值是类数组

  • 向左查找

    previousElementSibling：获取兄元素，唯一，如果没有，返回null

  • 向右查找

    nextElementSibling：获取弟元素，唯一，如果没有，返回null

<head>
    <meta charset="utf-8">
    <title></title>
    <style type="text/css">
        #tbl {
            border-collapse: collapse;
        }
        #tbl tr>td {
             border: 1px solid #ccc;
                padding: 10px 10px;
                text-align: center;
        }
    </style>
</head>
<body>
    <table id="tbl">
        <tr class="first">
            <td>1</td>
            <td>2</td>
            <td>3</td>
        </tr>
        <tr class="second">
            <td>4</td>
            <td class="ctr">5</td>
            <td>6</td>
        </tr>
        <tr class="third">
            <td>7</td>
            <td>8</td>
            <td>9</td>
        </tr>
    </table>
</body>
<script type="text/javascript">
    let ctr = document.querySelector(".ctr");
    //四向遍历
    //向上找父元素
    let tr = ctr.parentElement;
    console.log(tr.className);
        
    let table = tr.parentElement;
    if(table.tagName.toLowerCase() === "tbody") {
        table = table.parentElement;
    }
        
    //向下找子元素
    console.log(table.tagName);
    console.log(table.childElementCount);//子元素的个数
    console.log(table.children[0].childElementCount);
        
    let trs = table.children[0].children;
    for(let i=0;i<trs.length;i++) {
        let tr = trs[i];
        console.log(tr.className);
    }	
        
    //向前找兄元素
    let prev = ctr.previousElementSibling;
    console.log(prev.innerHTML);
        
    //向后找弟元素
    let next = ctr.nextElementSibling;
    console.log(next.innerHTML);
    
    let firstTd = document.querySelector("tr.first>td:first-child");
    
    console.log(firstTd.previousElementSibling);		
</script>
