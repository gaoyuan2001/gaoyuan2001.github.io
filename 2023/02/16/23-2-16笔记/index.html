css的三种引入方式

使用link标记

相当于，html专用于框架，css专用于装饰，各司其职

注意：

块元素：可以设置尺寸。独占一行

内联元素：大小取决于内容所决定的，由内容所撑起

内联元素不可以设置大小

  • 复合选择器：其中a和b都是其他选择器（基础选择器，或者其他复合选择器）

    a和b，既可以为标签名，也可以为样式名

    1.子代选择器：a>b，tr>td {……样式}，选择所有tr之下的td。tr是td的父

    2.后代选择器：a b，选择a下的后代b。a是b的祖先

    3.兄弟选择器：a~b所有弟，a+b紧邻弟

    • 所有弟选择器：crt~td {……样式}，选择与id = crt的所有的类型为td的兄弟元素们

    • 紧邻弟选择器：crt+td {……样式}，选择与id = crt的相邻的类型为td的兄弟元素们

    4.并集选择器：a，b

    • .crt，.l1 {……样式}

    5.交集选择器：ab，注意顺序，不要歧义

    • div.crt {……样式}

  • 超链接伪类选择器：爱恨准则。 love hate

    :link默认样式

    :visited访问之后的样式

    :hover鼠标划过的样式（不止可以用于a）

    :active激活的样式

  • 其他伪类选择器

    :first-child选中作为起父元素的第一个子元素的元素

    :last-child选中作为起父元素的最后一个子元素的元素

    **:nth-child(n)**选中作为起父元素的第n个子元素的元素

    :not(s)排除掉指定的选择器，其他的s都给设置样式

#tbl tr>td:last-child {
    background-color: gray;

#tbl tr>td:nth-child(2) {
    background-color: gray;

盒模型：box model！！！

css将任意一个html原色都视为一个盒子，以div为例

1、margin：外边距margin box

  • margin：30px 20px 10px 5px(上右下左，顺时针)，四个值
  • margin：30px 20px 10px（上，左右，下），三个值
  • margin：30px 20px（上下，左右），两个值
  • margin：30px（上下左右），一个值

也可以分别设置

  • margin-top上
  • margin-bottom下
  • margin-left左
  • margin-right右
<style>
    .my {
        width: 300px;
        height: 200px;
        background-color: pink;
                
        /* 外边距 */
        margin: 30px; 
        margin: 30px 20px;
        margin: 30px 20px 10px;
        margin: 30px 20px 10px 5px; 
                        
        margin-top: 30px;
        margin-bottom: 20px;
        margin-left: 10px;
        margin-right: 5px; 
    }
</style>

2、border：边框样式border box（width，style，color）

  • border-width：30px 20px 10px 5px边框宽度
  • border-style：solid solid dotted dashed边框样式
  • border-color：red blue white yellow边框颜色

也可以分别设置

  • border-top上
  • border-bottom下
  • border-left左
  • border-right右
<style>
    .my{
        /* 边框 */
        border: 30px solid #000;		
                
        border-width:30px 30px 20px 5px;
        border-style: solid solid dotted dashed;
        border-color: #000 #0f0 #f00 blue;
                
        border-top-color: #0f0;
        border-left-width: 2px;
                
                
        border-top: 30px solid #000;
        border-bottom: 10px dotted red; 
        }
</style>

3、padding：内边距 padding box

<style>
    .my{
        /* 内边距 */
        padding: 30px; 
        padding-left: 20px;
        padding-right: 50px; 
                
                
        /* 不独占一行的块元素 */
        display: inline-block;
        }
</style>

4、content：最里面是内容，content-area，width height是给内容设置的

style中默认设置尺寸，设置的就是内容区域

常用样式

1.border：边框样式

  • border: 1px solid blue; 边框

  • solid实线，dashed虚线 ，dotted点线

2.border-radius：边框圆角

3.border-collapse：collapse：边框折叠

#tbl {
    border-collapse: collapse;
}
/* 用id=“tb1”调用 */

4.font-weight：字重。normal正常，bold边框，lighter高亮，bolder加粗

5.display：显示，none（不显示），inline-block（行内块），block（块），inline（内联）

6.margin：外边距

7.padding：内边距

8.border-width：边框宽度

9.border-style：边框样式

10.border-color：边框颜色

11.visibility：hidden，隐藏了

12.text-decoration：设置a中的链接下划线，none

13.box-sizing：控制尺寸施加在哪个盒子上，border-box包含边框，content-box在内容区域外面

14：magin：0 auto，自动调整，居中

15.overfow：内容溢出，进行处理scroll，hidden

  • overfow-x
  • overfow-y

16.list-style-type：设置列表前的原点，none。

text-align：文本水平对齐

text-decoration： none;

ul是无序列表，全称是unordered list，先来个例子：
● 张三
●李四
●王二
●刘五
ol是有序列表 ,全称是ordered list，同样举个例子：
1、张三
2、李四
3、王二
4、刘五

这就是ul与ol的区别。
ul与ol前的符号是可以修改的。只需要修改它们的type值。
ul的type属性有：disc—实心圆(默认）、circle—空心圆、square—实心方块
ol的type属性有：1—数字（默认）、a—小写字母、A—大写字母、i—小写希腊字母、I—大写希腊字母。
通过CSS也可以在将它们前的符号改为图片，这样列表显示就会更美观了。
ul 是块级元素可以定义宽和高 ,li表示行排列不能单独使用，需要在ul和ol里面，配合使用。

浮动：是一种可以破坏文档流规则的机制。

通俗来说就是将这一元素浮起来，将会覆盖其之下的元素

浮动float：left|right

清除clear：

clear：both

1.必须使用一个兄弟元素来清除浮动

2.必须在浮动元素的后面

3.必须是块元素
