前端

浏览器：browser

专门用于解析 html、css和JavaScript的。是一款软件，应用级软件

Chrome及开源版本：Google，V8引擎

Firefox：Mozilla

IE及edge：微软

safari：Apple

opera：某一国家。360收购之后，废弃

前端三大件：没有竞品，不需要可虑过时

html：1.0、4.01、5

css：Cascading Style Sheets，层叠样式表

js：JavaScript

html：网页的框架结构

css：美化，修饰

JavaScript：负责动态效果，行为交互

html：超文本标记语言

html是由一个个标记组成的

html的标记是一个树形结构，一个标记可以嵌套着另外多个标记，一层一层

html标记有两种：

单标记：

<meta charset="utf-8">

双标记：标记开始和结束，成对出现，开始标记<>，结束标记</>

<html>
    <body>
        <div>
            <tr><td></td><td></td></tr>
        </div>
        <div>
            <tr><td></td><td></td></tr>
        </div>
        <div>
            <td></td>
        </div>
    </body>     
</html>

开始标记<>，结束标记</>，里面的内容称为一个元素

不管多少空格，在页面上只占一个英文半角的字符大小

块标记：独占一行，h，p，div，ul，ol，dl

内联标记：b，span

标记属性

size="50" color="blue" width="100%" align="center"

每个标记都有自己的属性，有些可以通用

  • align：水平

  • size：大小

  • color：颜色

  • width：宽度，可以用百分比，或者定值（单位px，像素）

CSS：casade style sheet，层叠样式表

负责样式的美化和装饰

css版本：1.0、 2.0、3.0

样式的三种引入方式：

  • 1.行内样式（内联样式）:直接在标记上使用style属性引入

    缺点：只能修饰当前所在标记

  • 2.页内样式：在当前页引入样式，供本页使用

    需要使用css选择器指定要修饰的元素集合

    指定修饰谁，通过选择器

    基础选择器：

    • 1.标记选择器：标记名{……样式}

    • 2.id选择器：定义样式：#id名称{……样式}，调用id=“pink-div”

    • 3.类选择器：定义样式：.类名称，标记内类名称定义：class=“类名 对象名”，一个标记可以有多个类名称，用空格分隔

    • 全选选择器：*{……样式}，表示选择全部元素

      <head>
    <meta charset="utf-8">
    <title></title>
    <style type="text/css">
        h1 {
            color: #00f;
            font-size: 25px;
            background-color: pink;
        }
        #pink-div {
            text-align: center;
        }
        .my {
            text-decoration: underline;
            width: 400px;
            height: 200px;
            text-align: center;
            font-style: italic;
            font-family: 宋体, 黑体, 隶书, 微软雅黑;
        }
        * {
            display: none;
        }
    </style>
</head>
<body>
    <h1 style="color: #00f; font-size: 25px; background-color: pink;">今天好运气</h1>
    <h2>老狼请吃鸡</h2>
    
    <h1 id="pink-div">我也要粉色</h1>
    
    <h1 class="my your">中国智造</h1
        
    <h1>2222</h1>
    
    <h1 class="my red">3333</h1>
    
    <h1>4444</h1>
    
    <h1 class="my a">5555</h1>
    
</body>

样式语法：

样式名（用-分隔）：样式值

多个样式用；分号分隔

常用样式：

1.color：前景色

2.font-size：设置字号。单位可以修改px(像素)，em，rem，

3.background-color：设置背景色

4.text-align：left/right/center文本内容的水平对齐方式

5.text-decoration：underline下划线，none没有任何下划线

6.width：设置宽度，单位可以是像素或者百分比

7.height：设置高度，单位可以是像素或者百分比

8.font-style：italic，字体斜体

9.font-family：设置字体，‘宋体’,‘隶书’,‘黑体’,‘微软雅黑’

10.display：none，不显示。block|inline|inline-block|flex|table|table-cell
