ajax

ajax技术：页面异步无刷新的请求机制。创造了web2.0时代

如果没有使用ajax，每有一次请求，都会刷新页面

请求->服务器->servlet->执行代码，到数据库中查询->服务器把jsp渲染成页面->客户端浏览器解析

ajax需要通过js来实现，借助于jquery

js->tomcat->servlet

响应的json数据很小，只有数据，响应更快

ajax响应类型

resp.setContextType(“applaction/json;charset=utf-8”);

post请求

json：一种js的对象表示方式。

键值对

{

k1:v1,

k2:v2,

k3:v3

}

数组用 […] 表示

代表发送一个ajax请求
第一个参数是请求地址，第二个参数是请求携带的参数，第三个参数是回调函数，第四个参数是数据类型
和浏览器解析页面可以同时进行

url-pattern:

/*：匹配任意请求，但不包括静态文件

*.jsp：匹配任意jsp

/：匹配任意请求，包括静态资源文件
