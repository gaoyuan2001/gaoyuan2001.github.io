常用类：String，Object，Date，Calendar，LocalDate，基本数据类型的封装类

String类：

语法糖：”a” = new String(“a”);

1.值不可改变。天生线程安全，连更改都不可以，多个线程同时更改的冲突不会出现。

引用在栈里，对象在堆里

StringBuilder：值可以改变。线程不安全

StringBuffer：值可以改变。线程安全的

2.final修饰：不可被继承

3.引入缓存机制。可以缓存字符串常量

String s1 = "abc";
String s2 = "abc";
System.out.println(s1 == s2);//true

String s3 = new String("abc");
System.out.println(s1 == s3);//false
//new之后，引用肯定指向堆中的对象，并在缓存池中创建字符串常量对象
//此后，新的引用再次=字符串常量（而不是new，如果再new，则再堆中创建一个新对象，而缓存池中不在创建相同的字符串常量），则新引用指向缓存池中的字符串常量对象

例：

String a1 = "ja" + "va";//创建两个对象"ja","va",不创建"java"对象,虚拟机jvm启动时肯定有java
String a2 = "java";//不创建对象

String a3 = "计算机" + "语言";//创建三个对象，"计算机","语言","计算机语言"
String a4 = "计算机语言";//不创建对象

泛型的定义和使用：

定义在类上：使用尖括号写在类名后面。在类内部进行约束

public class Test1<A> {
   private A a;
   public Test(A a){
   this.a = a;
   }
   public A get() {
   return this.a;
   }
}

public static main(String[] args){
   Test1<Integer> a1 = new Test2<Integer>(10);
   a1.get();
}

定义在方法上：写在方法的返回值类型之前。在方法内部进行约束

public class Test2 {
    
    public static <B> B build(B b) {    
        return b
    }

    public static main(String[] args){	
        int a1 = Test2.build(10);    
        long a2 = Test.build(10L);
    }
}

异常处理

try：尝试执行，监视一段代码，异常出现后，抛出异常，之后的语句不再执行

catch(Exception e)：捕获异常，捕获到异常后执行

finally：不管是否捕获到异常都会执行

throw：创建一个异常对象，并触发

public static void sayHello(File f) throws FileNotFoundException {  

    if(!f.getname().equals("aaa.txt")){
        throw new FileNotFoundException("文件未找到");
   }

// 将一个文件拆分成n份，存储到相同的路径之下
package com.situ.chapter2;

import java.io.File;
import java.io.FileInputStream;
import java.io.FileNotFoundException;
import java.io.FileOutputStream;
import java.io.IOException;

public class Test1 {

    public static void main(String[] args) {
        File f = new File("d:/data.pptx");
        splitFile(f, 5);
    }

    // 将一个文件拆分成n份，存在到相同路径之下
    public static void splitFile(File file, int n) {
        long perSize = file.length() / n;// 每个拆分后的大小
        // long lastFileSize = perSize + len % n;// 最后一个的大小

        FileInputStream fis = null;
        FileOutputStream fos = null;
        try {
            fis = new FileInputStream(file);

            // 先读出前n-1份
            for (int i = 0; i < n - 1; i++) {
                fos = new FileOutputStream(new File(file.getParent() + "/" + file.getName() + "-" + (i + 1) + ".spl"));

                int minRead = (int)(perSize < 1024 ? perSize : 1024);
                long readed = 0;// 表示已读取了多少个字节

                byte[] b = new byte[minRead];

                while (true) {
                    int len = fis.read(b);
                    readed += len;
                    fos.write(b, 0, len);

                    if (readed == perSize) {
                        break;
                    }

                    int l = (int)(perSize - readed);
                    if (l < 1024) {
                        fis.read(b, 0, l);
                        fos.write(b, 0, l);
                        break;
                    }
                }

                fos.flush();
                fos.close();
            }

            // 把剩余全部读成最后一个
            fos = new FileOutputStream(new File(file.getParent() + "/" + file.getName() + "-" + n + ".spl"));
            byte[] b = new byte[1024];
            while (true) {
                int len = fis.read(b);

                if (len < 0) {
                    break;
                }

                fos.write(b, 0, len);
            }

            fos.flush();
            fos.close();

        } catch (FileNotFoundException e) {
            e.printStackTrace();
        } catch (IOException e) {
            e.printStackTrace();
        } finally {
            if (fis != null) {
                try {
                    fis.close();
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }
        }

        System.out.println("结束");
    }
}
