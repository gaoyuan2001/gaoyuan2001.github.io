JavaSE基础语法

2023/1/30 学习 Java

年前回顾

Java语法与程序逻辑

计算机语言（二进制指令）-> 汇编语言 -> C语言 -> 高级语言（Java（最适合web）、C#、php…）

JDK：Java开发工具包

JRE：Java运行时环境

IDE：eclipse，idea

1.标识符，关键字，常量，输入，输出

标识符：字母（大小写），数字，下划线，不能以数字开头，不能是关键字，可以是$符，可以是UTF-8字符

关键字：public，private，class，int，short，long等，代表特定的语法含义

变量：int a = 10；a为变量，10为常量。final int a = 10；此时的a也是常量

Java中：输入输出本质是流操作（IO）。System.out.println()；

2.基本数据类型：8大

byte：字节，1个字节，1byte=8bit（位），输入输出流常用，-128—127

short：2个字节，-32768—32767

int：4个字节，-21亿—21亿

long：8个字节

float：4个字节，精度6—7位，float范围大于long，因为按照科学计数法存储的，4字节其中有一部分用于保存10的次方

double：8个字节，精度6—17位，按照科学计数法存储的

char：2个字节，char目前仍然无法表达任意字符

boolean：true/false，官方没有明确规定占用几个字节

BigInteger：大整数，存储无限精度

BigDecinal：大实数，存储无限精度

3.运算符，表达式

算数运算符： /分子分母都为整数，结果也是整数，10%3.5=3

比较运算符：< > <= >= !=，结果为boolean类型

逻辑运算符：&& || ！：与 或 非，逻辑运算，更高效；& | ：逻辑运算和位运算，逻辑运算上不如前者高效。短路问题，

赋值运算符：=，+=，-=，*=，/=，%=；

位运算符：& 与 |或 ^非 <<左移 >>右移，用于加密解密，图像引擎，游戏引擎

条件运算符：a ? b : c，其中a是Boolean类型，a如果成立取b值，否则取c值

4.控制流

顺序：自上而下，逐行执行（内存中亦是如此）

选择：分支结构，if…单分支，if…else…双分支，并列if…else…多分支，嵌套if…else…分支，switch(case default break)

循环：while，do….while…，for，增强for循环（迭代集合）

5.数组，数据结构，一组数据。用一个变量来存储一组数据。

下标：0到数组长度-1，.length()

定义：int[] a = new int[10]; int[] b = new int[] { 1, 2, 3}; int[] c = { 1, 2, 3};

6.函数：

独立的代码段，可以解决某一特定类型的问题，可以返回值（也可以不返回）

7.面向对象

Java：纯粹面向对象的语言

C语言不面向对象，C++在C语言上增加了面向对象的部分，Java在C++的基础上减少了一些灵活性的部分（指针）

类：类型，也可以自己创建新的类。抽出共性归纳成类，有着相同的行为，或者提供相同的服务

对象：以类为模板，创建出的实例。运行时，new出来的，存在于内存中

面向对象的三大特性：如果是四大特性加上抽象

1.封装：对象是有一定状态（字段 field ）和行为（方法）的个体。通过访问修饰符可以控制对象内部成员的可见性

public protected 什么都不写 private
外部 1
子类 1 1
包内 1 1 1
类内 1 1 1 1

2.继承：单根继承结构（Object），只能继承自一个类，直接或间接继承自Object，业界认为Object没有父类，优于多根继承结构C++。

关键字：extends

①增加新的行为（属性、方法）

②重写父类的行为

重写：rewrite、Override。覆盖父类（祖先类）的方法，同函数名，同参数

重载：方法名相同，但是参数列表不同（个数不同，顺序不同，类型不同）

3.多态：对同一个引用，可能会出现不同的实现（不同的行为）

父类 a = new 子类();

父接口 b = new 实现类();

//这是个子类，继承自父类Animal
public class Tiger extends Animal {
    public void run() {
    } 
}

//上下不是一个类，下面是测试类

public static void main(String[] args) {
    run(new Tiger());
    
    Tiger b = new Tiger();
    run(b);

public void run(Animal a) {
    a.run();
}

使用父类的引用，指向子类的对象。当调用父类的方法，实际上执行的是子类对象的方法。

System.out.println();

此打印就是如此，代码为：public void println (Object obj) { }，再调用obj的toString()方法

延伸：定义一个接口的引用，指向一个实现类的对象

父接口 b = new 实现类();

抽象类：使用abstract关键字修饰的类。不能被实例化。可以没有抽象方法。除此之外，和普通类没有区别。

抽象方法：使用abstract关键字修饰的方法，抽象方法没有方法体。一个类如果有抽象方法，则必须为抽象类

某些方法无法提供具体的行为实现，或者不适合提供实现，或者不想提供实现，就可以写成抽象方法

接口：没有已实现的方法，全部为抽象方法。接口中的方法默认为public abstract。

约束一个类应该拥有的哪些行为。

使用一个接口的引用，指向一个实现类的对象

a instanceof B；判断a是否为B的实例，返回值为boolean类型

接口的特点：

1.类使用implements关键字来实现接口

2.一个类可以实现多个接口，

单继承多实现

3.接口可以继承其他接口，多个。使用extends关键字

内部类（不建议写，降低易读性）：一个类定义在类内部的类

1.成员内部类（非静态内部类）：内部类作为外部类的一个普通成员，可以使用访问修饰符控制可见性。持有一个指向外部类的引用，自由的访问外部类的成员
A a = new A();//必须要先有外部类的对象
A.B b = a.new B();//才能创建内部类的对象

2.静态内部类；在内部类加static，没有指向外部类对象的引用，只能访问外部类的静态成员

A.B b = new A.B();

3.局部内部类：定义在代码块中的类，有作用域和生命周期，比较有局限。很少见。

4.匿名内部类： 使用频率稍微高一点。适用于只需要创建一个对象的场景。借助于继承、或实现接口

A a = new A(){
    
}
//a是匿名内部类的对象，没有类的名字，只有代码块作为类的定义

定义匿名内部类的同时，创建匿名内部类的唯一的对象

构造方法：

1.和类同名，用于创建对象并初始化对象。无返回值。

2.可以重载。

3.可以调用其他构造函数，this()，必须是方法体的第一句。

4.如果不定义构造函数，则默认创建一个公有的，无参的空构造函数。

5.构造函数的第一句，一定是this(……)，或者是super(……)的调用，当super()无参时，可以省略。super(……)表示调用父类的构造函数。

6.可以使用访问修饰符。

7.类先执行静态成员（块）。执行构造函数时，先调用父类的构造函数（递归）。非静态块在构造函数第一句之后执行，也就是在super()之后执行。然后执行普通代码，执行用户代码

static和final关键字：

1.static：表示静态的，类的。使用static修饰的成员，无需先创建对象，通过类名可以直接进行访问。不属于某一个具体的对象

2.final：

修饰变量、类的属性，表示常量，不可改变，不可二次赋值。

修饰方法，表示方法不可改变，不可被重写。

修饰类，表示类不可改变，不可被继承。
