JavaSE基础语法

2023/1/7 学习 Java

五、集合：List，Set， Map

集合：存储 元素的容器

List：列表。可以存储有序的元素，且可以重复
Set：集合，狭义上的集合。元素不可重复，无序
Map：映射。存储键值对。key，value

1. List：列表，和数组很像

ArrayList：
ArrayList list = new ArrayList();// 动态数组
ArrayList <Integer> list = new ArrayList <Integer>();
//泛型<Integer>修饰限制ArrayList，后面的<Integer>可以省略
List是个接口：继承自collection接口
ArrayList是继承List的类

collection接口

常用方法：
add();向列表尾部增加一个值
add(int index,a);在列表指定的位置插入
addAll(subList a);批量添加，添加a，a不止List类型，只要是继承自List接口的类都可以
addAll(int index,subList a);在列表中指定位置批量添加a

set(int index,a);修改列表中第index的值，改为a

remove(int index);删除列表第index个元素
remove(Object a);删除列表中第一次出现的a

get(int index);获取列表第index个元素的值

size();获取列表长度

clear();清空列表

clone();浅克隆，复制了引用，对象是同一个对象

contains(a);判断列表里是否包含a，返回值是boolean类型
containsAll(List a);判断列表里是否完全包含另一个列表

indexOf();返回某个值在此集合中首次出现的位置
lastindexOf();返回某个值在此集合中最后出现的位置

isEmpty();列表判空

toArray();转换成数组

subList();

补充知识点：

数据类型：基本数据类型，引用类型
基本数据类型有八种，引用类型无限制
引用类型作为函数的参数：不是克隆（复制）对象，而是传递对象的引用（引用是对象在堆中的地址），所以形参所指向的对象的变化，会同样反映到实参所指向的对象
参数的传递是值传递，传递的是引用
堆栈里存放引用和基本数据类型，堆里放对象
引用指向对象，是对象的代码上的代表

引用类型作为函数的返回值：返回的是引用，而不是对象的克隆
