迭代器:一种循环遍历集合所有元素的方式，迭代器是一个对象

iterator();返回值类型是Iterator类型的对象

Iterator it = nembers.iterator();//多态，Iterator接口，定义对象it
while(it.hasNext()){//判断是否有下一个元素
　　Integer i = it.next();//取出下一个元素
}

特点：
1.遍历所有元素，无法只遍历部分
2.迭代器迭代时，无需索引，只能往一个方向迭代
3.安全，由于不涉及索引，所以不会出现指针越界等问题
4.适用场景：适合需要遍历全部元素的场景，不适合遍历局部，以及按照不同方向遍历的场景

iterator()函数可以返回一个集合的迭代器对象

迭代器语法糖（增强for循环）

迭代器语法糖，没有索引，和上面的一样，语法上简化简化
for(a:b):b是集合，a是迭代过程中的集合b的某一个元素 

for(Integer i：numbers){  
　　System.out.println(i);  
}
forEach:隐式迭代

直接调用，Consumer的accept方法，需要自己重写 

//匿名内部类  
Consumer<Integer> consumer = new Consumer<>() {  
　　@override  
　　public void accept(Integer t){  
　　　　System.out.println(t * t);  
　　}  
}  
numbers.forEach(consumer);//调用

以上可以简写为：

numbers.forEach(t->System.out.println(t * t));    
//把方法体写在括号里
流操作：隐式迭代

numbers.stream().forEach(t->System.out.println(t * t));

3.Map：映射（a->b）

Map也是一个接口
HashMap是具体实现类
存储　键值对　(key，value)
key无序，不重复，在一个key上多次重新put()赋值，是一次次的覆盖，值为最后一次put()

Map<String, String> map = new HashMap<>();
map.put(“stu001”,”张三”);//一次存两个，键和值
map.put(“stu002”,”李四”);
map.put(“stu003”,”王五”);

取得时候用key取值
map.get(“stu002”);

特点：key无序且不重复的
如何判断key重复：
先调用hashCode，判断是否相同，如果相同，则认为两个key重复。如果hashCode不同，则再次调用equals方法，判断是否相同，如果相同，则认为key重复

常用方法：
1.存储键值对
map.put( , )
2.获取值
map.get()
3.元素数量
map.size()
4.清空
map.clear()
5.克隆，浅复制
map.clone()
6.判断容器中，是否包含某个指定的key或value
map.containsKey();
map.containsValue();

六、异常：Exception

异常处理机制：Java语法，及异常类型

程序中出现非预期的情况，导致程序无法正常运行，数据出错，或者程序终止。

出现异常后，程序立即终止，异常位置之后的代码不再执行
由程序员去捕获异常，在捕获异常后，进行一定的补救或者挽回措施，同时保证程序可以继续运行
捕获私有前后顺序的。当出现异常时，只会进入一个分支
try　{ //程序

}　catch　(异常类型1)　{

}　catch　(异常类型2)　{

}　catch　(Exception　e)　{
　　e.printStackTrace();
}

所有异常类型，都直接或间接的是Exception的子类（后代类）
常见异常类型：
　　Exception：异常的父类
　　ArithmeticException：数学异常
　　RuntimeException：运行时异常
　　ArrayIndexOutOfBoundsException：数组下标越界
　　StringIndexOutOfBoundsException：字符串索引越界异常
　　IndexOutOfBoundsException：下标越界异常
　　NullPointerException：空指针异常 

try　{// 监控程序   
　　int　c　=　a　/　b;    
}　catch (ArithmeticException　e)　{// 捕获异常类型    
　　e.printStackTrace();//打印异常路由    
　　System.out.println("除数为0");   
}　catch (RuntimeException　e1)　{   
　　e.printStackTrace();   
　　System.out.println("运行时异常");   
}   
System.out.println("运行结束");
