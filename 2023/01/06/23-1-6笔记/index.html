JavaSE基础语法

2023/1/6 学习 Java

知识补充：

5.内部类

Java语法特性
用的不多
在一个类的内部，定义一个类

成员内部类（非静态内部类）：也是外部类的一个成员
使用需要先创建外类的对象，然后用对象.创建成员内部类
内部类中，持有一个其外部类对象的引用，可以访问外部类的成员
如果成员内部类继承于另一个类，则间接实现了多继承
静态内部类：相当于外部类的一个静态成员
只能访问外部类的静态成员

局部内部类：定义在局部块内的类，有作用域和生命周期，只能再块内使用，使用场景十分受限
匿名内部类：使用频率相对而言，会比其他内部类多一些
没有名字
不能定义构造函数
本质上是一种继承，或者实现接口，多态的一种体现
类的定义，与创建类的唯一对象，结合到一起的
定义类的同时，创建匿名内部类的唯一对象

Person p = new xxx extends Person() {// xxx extends省略，
匿名内部类，继承自Person类
};
Person p = new Person() {
匿名内部类
};//末尾加分号

五、集合：List，Set，Map

集合：存储 元素的容器

List：列表。可以存储有序的元素，且可以重复
Set：集合，狭义上的集合。元素不可重复，无序
Map：映射。存储键值对。key，value

1. List：列表，和数组很像

ArrayList：
ArrayList list = new ArrayList();// 动态数组
ArrayList <Integer> list = new ArrayList <Integer>();
//泛型<Integer> 修饰限制ArrayList，后面的可以省略
常用方法：
add();向数组尾部增加一个值
set(int index,a);修改index的值，改为a
remove(int index);删除第index个元素
get(int index);获取第index个元素的值
size();获取数组长度
