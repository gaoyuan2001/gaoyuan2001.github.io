JavaSE基础语法

2023/1/3 学习 Java

回顾：
　　Java语法基础
　　　1.变量
　　　2.数据类型：八种基本数据类型
　　　3.运算符
　　　4.表达式和语句
　　　5.流程控制结构：顺序、选择和循环
　　　6.数组：下标可以和循环结合使用
　　　7.函数：定义、调用（形参、实参，传参）、函数重载、递归调用

类型的构成
　　属性和方法（通过.来调用引用成员）
　　　成员属性：静态属性
　　　成员方法：动态属性，行为
　　封装：将数据与对象绑定在一起，并且控制数据的可见性

　　成员访问权限控制符
　　　public：公有的。任何地方都可以访问
　　　protected：受保护的。允许子类访问，允许同一个包下的类访问，类外不能访问
　　　空：什么都不写，包方位修饰符。仅允许当前包内的类访问，必须同属一个包下的
　　　private： 私有的。只能在当前类内部访问，外部不允许访问
　　如果不知道要定义什么访问权限，优先使用更小的的权限，安全，封装

　　　setter，getter方法
　　　　setter：设置属性值的公有方法
　　　　getter：获取属性值的公有方法(boolean类型的前面加is)
　　　　　this.表示当前调用

　　构造函数（构造方法）
　　　构造函数，是用来创建对象的，new后面
　　　1.和类同名
　　　2.如果用户没有手动创建构造函数时，会自动创建空的、公有的、无参的默认构造函数
　　　　　public void Student () {}
　　　3.如果用户手动创建构造函数时，则不再创建默认的构造函数
　　　4.构造函数：用于对对象的属性进行初始化
　　　　　区分默认值和初始化，默认值是在定义属性时给予的值，初始化是创建对象时构造函数对属性的赋值
　　　5.构造函数可以有参数列表，可以传参给属性
　　　6.构造函数可以重载
　　　7.构造函数中可以访问属性，也可以调用方法，this.
　　　8.构造函数可以调用其他构造函数，使用 this() 调用，当前类的构造函数
　　　　　注！必须要放到第一行

　　对象成员属性的默认值
　　　1.引用类型默认值时null
　　　2.int：0
　　　3.boolean：false

面向对象有三大特性：封装、继承、多态
　　封装：属性和方法结合起来

　　继承：extends子类在父类基础上增添了其他的属性或者方法，复用一些重复的属性和方法（子类 extends 父类）
　　　　　注：公有public、受保护protected可以被继承，但是构造函数不能被继承，无修饰符和私有成员private不能被继承

快捷键：ctrl+shift+O 快速导入包

　　　　　调用父类的构造函数 super(); 来对父类属性进行初始化
　　　　　子类必须要先调用父类的构造函数，然后才能执行自身的构造函数
　　　　　　　this()调用类自身的构造函数，super()调用父类的构造函数

　　　　　继承的注意事项：
　　　　　　　1.继承是可以递归继承的
　　　　　　　2.一个类只能继承自另一个类，也就是只能有一个直接父类，不支持多重继承，单根继承
　　　　　　　3.如果一个类没有手动继承自另一个类，默认继承自Object类，Object是终极父类,只有Object类没有父类
　　　　　　　4.任何一个类，都直接或间接的继承自Object类
　　　　　　　5.任一构造函数，都需要先（直接或间接）调用父类的构造函数。构造函数第 一句要么使用super()调用父类构造函数，要么使用this()调用本类构造函数。如果什么都没有写，则默认调用无参super()，此时的无参super()调用的是父类无参的构造函数，如果父类没有无参的构造函数，则报错

　　　　　super
　　　　　　　1.super. 可以调用父类的属性和方法（区别this. 调用本类的方法）
　　　　　　　2.super() 调用父类构造函数,必须放在第一行

　　　　　重写方法：在子类中，方法名和父类一样，此时对父类继承下来的方法进行覆盖掉 @override
　　　　　　　　　　　　调用时，会调用子类的方法，而不是父类的方法，如果要调用父类的方法，用super.调用
　　　　　　　　　　　　子类重写的方法，访问权限不能缩小

　　　　对象与对象之间的三种结构关系
　　　　　　1.聚合关系：has a 包含关系 例：人的对象 包含 名字对象
　　　　　　2.依赖关系：use a 使用关系 一个对象在动态行为过程中，使用到了另外其他的对象 例：学生A的学习这个行为对象 用到 电脑对象
　　　　　　3.继承关系：is a 例：学生对象 是 人对象

　　多态：一个对象所展现出多种形态
　　　　　Person p1 = new Student();
　　　　　Person p2 = new Teacher();
　　　　　把p1看成人，而不是学生，或者把p2看人，而不是老师
